Nombre del juego

MODO DE JUEGO:

• Jugador vs PC.

OBJETIVO DEL JUEGO:

• Descifrar el número de la PC.

INSTRUCCIONES:

• Juego por RONDAS

• A la PC se le asigna un numero random constante de 4 cifras, el cual no puede tener dígitos repetidos ni puede comenzar con 0. Ese es el número que el jugador intentará descifrar. Para ejemplificar el desarrollo, el jugador será mencionado como PL y el número de la PC será indicado como NPC.

Ejemplo: NPC = 1506

DESARROLLO DEL JUEGO:

• Empieza la primera ronda (Ronda = 1). El jugador debe ingresar un número de 4 cifras (indicado como N en el ejemplo) que será comparado con el número de la PC (ejemplo: N = 1540). La comparación se realiza de la siguiente forma:

o Si hay dígitos de N que estén incluidos en NPC (n dígitos), se verifica si esos dígitos se encuentran en la posición correcta con respecto a las posiciones de los dígitos de NPC. Si hay dígitos que estén correctamente posicionados (x dígitos), se dice que N tiene n - x mal y x bien. Si n < 4, se sobrentiende que hay 4 - n dígitos de N que no están contenidos en NPC. Se omite mostrar esa información para generar un mayor nivel de interpretación del jugador.

Ejemplo:

	NPC = 1506
		 
	N = 1540
		 
	n = 3 (1, 5 y 0 están incluidos en NPC)
		 
 	x = 2 (1 y 5 están correctamente posicionados)
		 
	Dígitos de N que no están contenidos en NPC: 4-n = 4-3 = 1

• Finaliza la primera ronda y continúa el juego, con la idea de que el jugador utilice la información de rondas anteriores para descifrar el número de la PC.

• Al finalizar cada ronda, se mostrará una tabla con los siguientes datos: N° de ronda (Ronda), número a comparar (N), dígitos “bien” (x), dígitos “mal” (n - x).

• Cuando el número introducido sea igual a NPC, finaliza el juego.
